## 二、为什么12-1有效？

### 理论机制验证

#### 1. 短期反转效应（A股显著）
```
1月涨幅 vs 未来1月收益 相关系数 = -0.0898
→ 涨得多的下月跌（反转）
→ 剔除最近1月避免买入短期涨幅过高标的
```

#### 2. 中长期趋势延续（A股较弱）
```
前11月涨幅 vs 未来1月收益 相关系数 = -0.0344
→ 中长期动量效应较弱（接近随机）
→ 但剔除短期反转后仍能捕捉趋势
```

#### 3. 回撤显著改善
```
12-1截面 MDD: -29.2%
12月截面 MDD: -40.9%
改善: 11.7个百分点
```

## 三、截面动量 vs 时序动量

### 截面动量（排名选股）
- 逻辑：选过去11月涨幅最大的ETF
- 适用：资产池内轮动（多标的）
- A股表现：Sharpe 0.54~0.71 ✓

### 时序动量（趋势跟踪）
- 逻辑：单标的动量>0持有、<0空仓
- 适用：单资产择时
- A股表现：Sharpe 0.34~0.39 ✗

**结论：A股ETF更适合截面轮动而非时序择时**

**RSRS v4更优原因**：
1. C63复合动量（50/63/75日三周期平均）比单一12-1更稳定
2. RSRS大盘择时过滤掉了熊市亏损期
3. 波动率仓位管理控制回撤

**但12-1动量仍有价值**：
1. 逻辑简单透明（剔除短期反转）
2. 月频调仓降低交易成本
3. 可作为多因子组合的一个信号
