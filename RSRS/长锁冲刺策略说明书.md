# RSRS × C63 长锁冲刺策略 (v2.0)

## 策略定位

基于 RSRS 大盘择时 + C63 横截面动量选股，配合**长锁仓机制**减少毛刺出场，追求高复合增长。属于**趋势跟踪 + 动量选股**的复合体系。

## 策略三层逻辑

### 第一层：RSRS 大盘择时（何时进场）

使用沪深300的 **阻力支撑相对强度（RSRS）** 指标判断大盘趋势：

```
N = 18          # 回归窗口（18个交易日 ≈ 1个月）
M = 1200        # z-score窗口（1200个交易日 ≈ 5年）
Buy = 0.7       # z-score ≥ 0.7 开多
Sell = -1.0     # z-score ≤ -1.0 平仓
```

- 当 z-score > 0.7：**买入信号**
- 当 z-score < -1.0：**空仓信号**
- 当 -1.0 ≤ z-score ≤ 0.7：**维持当前状态**

### 第二层：长锁仓机制（核心创新）

**传统 RSRS 的问题：** z-score 频繁在 0.7 附近震荡，造成"开多→转空→再开多"的毛刺损耗。

**锁仓机制：** RSRS 给出买入信号后，**锁定做多状态至少 84 个交易日（约 4 个月）**，期间即使 RSRS 短暂转空也忽略。

```
Lock = 84 交易日
```

效果：
- ✅ 减少 50% 以上的虚假出场
- ✅ 捕获大趋势的完整涨幅
- ⚠️ 遇到深度熊市会吃回调（但 84d 内 RSRS 重新转多的概率高）

### 第三层：C63 动量选股（买什么）

在每个调仓日，计算池中全部 7 只 ETF 过去 63 个交易日的收益率：

```
Score(code) = ret_63(code)   # 单一63日窗口
```

选择得分最高的 ETF，满仓持有。

```
调仓周期(RB) = 63 交易日
持仓只数(Top) = 1
动量过滤：只选正收益
```
