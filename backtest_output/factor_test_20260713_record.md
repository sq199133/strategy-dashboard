# A股ETF多因子回测结果记录

## 任务
对281只A股ETF进行因子回测，包括单因子测试和多因子组合，输出结果及分析。

## 输出目录
D:\QClaw_Trading\backtest_output\factor_run_20260713_230732\

## 回测设置
- 数据：D:\QClaw_Trading\data\history\ (281只ETF，272只有效)
- 回测范围：2021-01 ~ 2026-07 (2019起暖机24个月)
- 频率：月末截面排序 → 次日开盘执行 → 等权持有 → 次月再平衡
- 手续费：买卖各0.03%，ETF免印花税
- 参数：TopK = 5/10/20，共测试14个单因子×3 + 7个多因子组合×3 = 63种配置

## 核心发现

### 1. 低波动因子是最强单一因子
低波动12个月窗口、Top5持仓：年化11.7%，Sharpe 5.89，最大回撤-4.9%，胜率73.9%
这是全表最优结果，但有重大偏差（见下）。

### 2. 黄金ETF污染
2021-2026年黄金ETF 518880涨幅114.6%，同期沪深300 -8.9%。低波动因子持续选中黄金ETF。
- 低波动12m top5：76%利润来自黄金ETF（5只持仓中3只黄金）
- 低波动6m top10：47%来自黄金
- 低波动3m top5：37%来自黄金（最均衡）

### 3. 排除黄金后的真实表现
低波动3m top5 剔除黄金后年化约6-8%，Sharpe约1.0-1.2，月胜率62%。
持仓偏向：QDII ETF(标普500/纳指)、红利ETF、可转债ETF、债券ETF — 逻辑自洽。

### 4. 动量因子完全失效
动量12-1m top5 年化-3.1%，3m动量-15.9%。所有动量变体无正Sharpe。
A股ETF(行业ETF为主)趋势持续性弱于美股个股。

### 5. 多因子组合
动量+低波组合 Sharpe 4.74(年化11%)，但黄金贡献117%(动量选的拖后腿了)。
低波+量能趋势组合在排除黄金后相对干净(50%黄金)，Sharpe 3.11，年化4.2%。

### 6. 最佳实战方案
低波动3个月窗口 → Top5 → 月频再平衡
- 预期年化：6-8%(排除黄金)
- 预期最大回撤：<6%
- 选股逻辑：买过去3个月波动率最低的5只ETF，等权持有
- 底层实质：黄金/美股QDII/债券/红利ETF组合
