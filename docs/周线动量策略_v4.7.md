# 周线动量策略 v4.7

> **版本**: v4.7 | **日期**: 2026-07-12  
> **核心改进**: MA21 趋势硬过滤 + c_bonus 归零  
> **验证**: OOS 年度回报 2023-2026，4/4 年跑赢旧策略

---

## 策略概述

基于 ETF 周线动量的趋势追踪策略。每周扫描全市场 ETF 池，依据动量因子排序，顶仓持有 1 只，动态再平衡。

---

## 核心参数

| 参数 | 值 | 说明 |
|:---|:---:|:---|
| `top_n` | **1** | 每周最多持有 1 只 |
| `max_dev` | **20%** | 价格相对 MA21 最大偏离度 |
| `SCORE_W1` | **0.50** | 近端动量权重（1 周） |
| `SCORE_W3` | **0.50** | 中端动量权重（3 周） |
| `SCORE_W8` | **0.00** | 远端动量权重（8 周） |
| `ATR_FACTOR` | **0.85** | ATR 波动率过滤阈值 |
| `LB` | **3** | 中端动量回看周期（周） |
| `VOL_SKIP` | **1.5** | 放量上限（量比） |

## 候选条件（筛选层）

ETF 必须通过以下全部条件，方可进入打分排序：

```
1. 赛道去重  — 同赛道仅取分最高者
2. ATR过滤  — ATR_atr > 0.85（波动率足够）
3. 偏离过滤  — |price / MA21 - 1| < 20%
4. 缩量过滤  — 当周量比 < 1.5（量能收缩）
5. MA21硬过滤 — price > MA21  ← v4.7 核心新增
```

### MA21 硬过滤说明

旧版本将 MA21 作为软加分项（c_pat 条件中 `ci > ma5 > ma21`），新版本将 MA21 **提升为硬门槛**：

```python
# 候选硬条件（必须全部满足）
if price <= ma21:   # 价格必须在 MA21 之上
    continue         # 直接排除，不进入排序
```

**为什么有效**：  
MA21 是中期成本均线。价格在 MA21 之上代表中期资金处于盈利状态，该状态下趋势延续概率更高。OOS 验证（2023-2026）证明该过滤贡献了核心超额。

## 动量打分公式

```
SCORE = 0.50 × mom_1w + 0.50 × mom_3w + 0.00 × mom_8w

其中：
  mom_1w = price / price_1w_ago - 1   （近端动量）
  mom_3w = price / price_3w_ago - 1   （中端动量）
  mom_8w = price / price_8w_ago - 1   （远端动量，已归零）
```

## 交易规则

### 建仓
- 每周最后一个交易日收盘后扫描，信号在**下周一**开盘执行
- 目标仓位：100% 资金，单只 ETF

### 止损 / 止盈
| 条件 | 规则 |
|:---|:---|
| 硬止损 | 单只浮亏 ≤ -8% → 止损出场 |
| 回撤止损 | 从高点回撤 ≥ 10% → 止损出场 |
| 换仓 | 有更高打分 ETF → 卖出当前，买入新标的 |

### 仓位管理
- 使用 ATR 动态权重：`weight = total_assets × (atr_i / Σatr) × 2`
- ATR 权重下限：0.3（防止极端低波 ETF 主导仓位）
