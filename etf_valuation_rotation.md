# ETF估值轮动策略 — 策略文档

> 策略框架：BaoStock全量ETF数据驱动 | 估值分位 + 动量 + ATR 三维因子轮动

---

## 一、数据来源与处理

### 1.1 数据接口
- **来源**：BaoStock `query_daily_history_k_ETF(date)` 接口
- **数据范围**：2024-01-01 ~ 2026-07-10（每5个交易日取一次，减少请求量）
- **字段**：
  ```
  date, code, open, high, low, close, preclose, volume, amount,
  pctChg, peTTM, pbMRQ, psTTM, pcfNcfTTM, isST
  ```
- **存储**：每日数据存为 `D:\QClaw_Trading\data\baostock_etf\{YYYYMMDD}.csv`

### 1.2 数据验证
- 单日最多约 **1594 条** ETF记录（2026-07-10实测）
- peTTM / pbMRQ 大量为空值，仅宽基、金融、消费等A股相关ETF有估值数据
- 跨境QDII（纳斯达克、德国等）通常无PE/PB

### 1.3 预处理
- 过滤 `isST=1`（ST股）
- 过滤 `volume=0`（停牌ETF）
- peTTM/pbMRQ为0或空时视为无效
- BaoStock code格式：`sh.510010`（沪）/ `sz.159001`（深）

## 二、板块指数构建

### 2.1 ETF池
- 读取 `etf_pool_V1_full.json`，共196只ETF，覆盖：
  ```
  宽基A股 | 红利策略 | 商品/周期/资源 | 金融 | 消费
  医药生物 | 科技/TMT/AI | 芯片半导体 | 制造/基建/公用
  港股/中概 | 科创板 | 新能源 | 跨境QDII-海外 | 其他
  ```

### 2.2 板块聚合
- **等权平均**：同板块内所有ETF收盘价取平均作为板块价格指数
- 板块日收益 = 今日板块均价 / 昨日板块均价 - 1
- 板块PE/PB = 成分ETF的PE/PB均值（仅用有效值）

### 2.3 ATR计算
- 单只ETF TR = High - Low（日内波幅）
- 板块ATR = 成分ETF TR的均值

## 三、因子构建

### 3.1 动量因子（12天）
```
动量 = Σ(过去12个交易日板块日收益率)
```
- 12天约两周，反映中期趋势惯性
- 基准每5天取样，动量窗口内实际约含2~3个数据点

### 3.2 ATR百分位（60天滚动）
```
ATR分位 = 当日ATR在近60天ATR中的百分位排名
```
- 过滤低波动板块（动量信号在低波环境容易失效）
- ATR分位 < 0.10 的板块排除

### 3.3 估值分位（近5年滚动窗口）
```
PE分位 = (近5年PE ≤ 当前PE的数量) / 有效PE数量
PB分位 = (近5年PB ≤ 当前PB的数量) / 有效PB数量
估值得分 = (PE分位 + PB分位) / 2
```
- **低估值**：估值得分 < 0.3
- **中性**：0.3 ≤ 估值得分 ≤ 0.7
- **高估值**：估值得分 > 0.7

## 四、四大策略

### 策略A — 纯动量TOP1（基准策略）
```
选股范围：全部板块（ATR分位≥0.10过滤）
选择规则：12天动量最高的板块
调仓频率：每5个交易日（约每月）
```
- **逻辑**：动量是最强单因子，ATR过滤避免低波假信号
- **优势**：简单高效，长期有效
- **劣势**：高估值板块可能已在高位，接盘风险

### 策略B — 低估值过滤 + 动量
```
第一步：筛选估值得分 < 0.4 的板块（低估值板块池）
第二步：在低估值板块池中选动量最强
若低估值池为空，则放宽至全市场动量
```
- **逻辑**：在便宜的地方追趋势，降低估值泡沫破裂风险
- **优势**：估值保护，减少追高
- **劣势**：低估值的板块可能长期低估（价值陷阱）

### 策略C — 估值加权（连续变量）
```
动量分数 = (动量 - min) / (max - min)  标准化到[0,1]
综合分数 = 动量分数 × (1 - 估值得分)
选综合分数最高的板块
```
- **逻辑**：不是二元过滤，而是让低估值板块获得更高的动量权重
- **优势**：连续变量，不损失信号；低估值板块有天然加分
- **劣势**：高动量+高估值的科技板块可能被压制

### 策略D — 纯估值（反向测试）
```
选估值得分最低的板块，不考虑动量
```
- **逻辑**：检验估值单因子的独立预测能力
- **预期**：作为对照，若年化为负，说明纯价值投资在A股ETF轮动中失效

### 5.1 Demo回测结果（模拟数据）

> 注：以下为Demo模式结果，使用模拟数据验证策略逻辑。真实数据运行后结果可能显著不同。

| 策略 | 总收益率 | 年化收益 | 最大回撤 | Sharpe | Calmar | 期末净值 |
|------|---------|---------|---------|--------|--------|---------|
| 基准（等权宽基）| +26.4% | 3.7% | -15.9% | 0.740 | - | 1,263,968 |
| **策略A: 纯动量TOP1** | **+64.6%** | **7.9%** | -37.5% | **0.921** | 0.212 | **1,645,835** |
| 策略B: 低估值过滤 | +12.8% | 1.9% | -35.7% | 0.236 | 0.052 | 1,128,192 |
| 策略C: 估值加权 | +29.6% | 4.1% | -27.3% | 0.507 | 0.148 | 1,295,855 |
| 策略D: 纯估值 | +17.5% | 2.5% | -25.5% | 0.304 | 0.098 | 1,174,617 |

**Demo结果分析：**
- **策略A（纯动量）** 遥遥领先，年化7.9%，Sharpe 0.92，但最大回撤37.5%较高

- **策略C（估值加权）** 介于两者之间，较好地平衡了回撤（-27.3%）和收益
- **策略D（纯估值）** 收益一般，但回撤相对可控

## 六、预期结果分析框架

### 6.1 核心假设检验
1. **估值有效假设**：低估值板块长期表现优于高估值 → 策略B/C应跑赢策略A
2. **动量有效假设**：12天动量对A股ETF板块有预测力 → 策略A应跑赢基准
3. **估值+动量协同**：连续加权策略C最优 → 期待Sharpe最高
4. **价值陷阱风险**：纯估值D最差 → 证实A股板块低估值≠均值回归

### 6.2 风险点
- **区间选择偏差**：2020-2026含疫情复苏+AI行情，可能放大科技板块动量
- **PE/PB数据缺失**：大量ETF无估值，板块覆盖不全
- **每月调仓**：交易成本低但可能错过短期拐点
- **BaoStock数据延迟**：peTTM/pbMRQ非实时，可能滞后1~2天

### 6.3 参数敏感性
- 动量窗口：5天 / 12天 / 20天（短期 vs 长期动量）
- 估值阈值：0.3 / 0.4 / 0.5（保守 vs 激进）
- ATR过滤阈值：0.05 / 0.10 / 0.20
