# 买卖信号速查表

## 🟢 买入信号（全部满足才开仓）

```
条件1: 收盘价 > MA20          ✓ 站上均线
条件2: MA20 斜率 ≥ 0          ✓ 均线走平或向上
条件3: MACD 金叉 OR 0轴上红柱放大  ✓ 动能确认
条件4: 成交量 不明显萎缩       ✓ 量能配合
```

### 信号强度排序

| 强度 | 组合 |
|------|------|
| ⭐⭐⭐ 最强 | MA20 向上 + 0轴上方金叉 + 温和放量 |
| ⭐⭐ 强 | MA20 向上 + 0轴下方金叉 + 温和放量 |
| ⭐ 一般 | MA20 走平 + 金叉 + 缩量（谨慎） |

### 买入时机
- 满足条件当日尾盘 14:50 后
- 或次日开盘竞价

---

## 🔴 卖出信号（任一触发即离场）

| 优先级 | 信号 | 动作 |
|--------|------|------|
| P0 | 收盘价跌破 MA20 | 次日开盘卖出 |
| P0 | 亏损 ≥ 5% | 无条件止损 |
| P1 | MACD 死叉 | 减仓 50% |
| P1 | 红柱明显缩短转绿 | 减仓 50% |
| P2 | 盈利 10%~15% | 可减仓/止盈 |
| P2 | 盈利 20%（主升浪） | 移动止盈 |

---

## ⚠️ 不做的信号（过滤）

- MA20 下行 + 任何金叉 → 不做（逆势）
- MA20 走平 + MACD 缠绕 → 不做（横盘）
- 短期暴涨后高位死叉 → 不做（接飞刀）
