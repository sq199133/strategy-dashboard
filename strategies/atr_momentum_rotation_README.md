# ATR动量板块轮动策略

## 一句话

> 每周一，从9个板块中选出**过去12天涨幅最大**、且**波动率未萎缩（ATR比>0.85）**的板块全仓买入。没有符合条件的板块就空仓。

## 策略参数

| 参数 | 值 | 说明 |
|:---|---:|:---|
| 动量窗口 | 12天 | 短期趋势强度 |
| ATR比阈值 | 0.85 | ATR(14)/ATR(60)，确认波动未萎缩 |
| 持仓数量 | TOP 1 | 集中持仓最强板块 |
| 换仓日 | 周一 | 每周第一个交易日 |
| 空仓收益 | 年化2% | 货基参考 |
| **硬止损** | **5%** | **从入场价跌超5%即平仓，跳过下周一的换仓** |

## 操作流程

```
周五收盘后：
  计算9板块的12日涨幅 + ATR比
    ↓
周一早盘（或节后第一交易日）：
  1. 剔除ATR比 ≤ 0.85 的板块（波动萎缩）
  2. 从剩下板块中选12日动量最强的1个
  3. 如果和上周持仓不同 → 卖出旧仓位，买入新板块
  4. 如果没有板块中选 → 空仓（货基）
```

## 止损规则（2026-06-10新增）

- **硬止损5%**：持仓后，板块指数从入场价下跌超5% → 立即平仓转现金
- **跳过下周一**：止损触发后，**跳过下一次周一换仓**，等下下周一再重新入市
  - 例如：周三触发止损 → 空仓跳过下周一 → 下下周一正常换仓
  - 为什么跳过？刚止损说明动量已断，等一周确认趋势更安全
- **效果**：最大回撤从-40.94%降到约-18.6%，年化反而提升到~29%

为什么止损后年化反而提高？因为5%止损在大跌中及早抽身，保住本金参与后续反弹，避免了深度套牢后的负复利效应。

## 注意事项

1. **2015年亏-8.44%（无止损）/ 有止损后改善**：股灾全面崩盘，止损会在-5%时出局
2. **2018年亏-13.32%（无止损）/ 有止损后减轻**：单边熊市中多次触发止损
3. **V型反转是死穴**：暴跌后立即暴涨，止损后可能错过快速反弹（2019年曾跟住HS300，但止损后不确定）
4. **止损不是万能**：在急涨急跌的震荡市中可能被频繁止损摩擦
