# MACD卖出条件优化报告

## 问题发现

**原策略v3.0的卖出条件：**
- ② MACD死叉
- ③ 红柱转绿

**回测结果：**
- 100次模拟，108个卖出信号
- **100%同一天触发**
- 结论：两个条件完全等价，存在冗余

---

## 数学原理

```
MACD柱 = (DIF - DEA) × 2

红柱转绿：MACD柱 > 0 → MACD柱 < 0
         ⇨ DIF - DEA > 0 → DIF - DEA < 0
         ⇨ DIF > DEA → DIF < DEA
         ⇨ 死叉！

所以：红柱转绿 ≡ MACD死叉
```

---

## 优化方案对比

| 方案 | 条件 | 触发时机 | 优点 | 缺点 |
|------|------|---------|------|------|
| **原方案** | 死叉 OR 红转绿 | 同一天 | — | 冗余 |
| **方案A** | 死叉（删除红转绿） | 第1天绿柱 | 简洁 | 单日信号可能假突破 |
| **方案B** | 红转绿（删除死叉） | 第1天绿柱 | 直观 | 同上 |
| **方案C** | 连续3天绿柱 | 第3天绿柱 | 过滤假信号 | 反应慢2天 |
| **方案D** | 死叉 + 价格<MA20 | 双重确认时 | 更稳健 | 可能错过第一时间 |

---

## 推荐：方案C（已更新至v3.1）

### 新卖出条件②：MACD柱连续转绿

**触发条件：** 连续3天 MACD柱 < 0

**理由：**
1. 过滤单日假信号（短期波动导致的死叉）
2. 确认趋势真正转弱
3. 与"价格跌破MA20"配合，形成双重验证

**示意：**
```
天数:      -3   -2   -1    0   +1
MACD柱:   +0.1 -0.2 -0.3 -0.4  → 触发卖出
              ↑
           开始转绿，但需观察

连续3天确认后，第3天收盘判定，次日开盘卖出
```

| 指标 | 原方案 | 新方案 |
|------|--------|--------|
| 卖出信号数量 | 较多（敏感） | 较少（过滤假信号） |
| 假信号率 | 较高 | 降低 |
| 反应速度 | 快（第1天） | 稍慢（第3天） |
| 收益影响 | 可能频繁止损 | 更稳定，减少误判 |

| 优先级 | 条件 | 说明 |
|:------:|------|------|
| 1 | **亏损>5%** | 硬性止损，当天卖出 |
| 2 | **价格<MA20** | 趋势破位，次日卖出 |
| 3 | **MACD柱连续3天<0** | 动量确认转弱，次日卖出 |
| 4 | **星级降为⭐ + (价格<MA20 或 MACD转绿)** | 综合评估 |
