## 一、联动总原则

```
盘后信息（新闻/政策/数据）
        │
        ▼
  ┌─────────────┐
  │  分类判断   │  ← 按本章规则判定等级
  └─────────────┘
        │
        ▼
  ┌─────────────┐
  │  联动操作   │  ← 对应信号/持仓的具体调整
  └─────────────┘
        │
        ▼
  ┌─────────────┐
  │  记录+执行  │  ← 写入复盘报告 + 复盘日次日执行
  └─────────────┘
```

**时间窗口：**
- **21:00 复盘**：评估当日收盘后 ~ 21:00 之间发布的A股/港股相关信息
- **次日 09:15 前**：评估隔夜美股/国际重大信息，复盘日次日开盘前微调

## 二、信息等级判定规则（自动触发）

### 🔴 重大（否决级）— 触发立即行动

**判定条件（满足任一）：**

| 类型 | 触发条件 | 影响范围 |
|------|---------|---------|
| 贸易战 | 中美互加关税 ≥ 10%，或突发贸易限制公告 | 全市场 |
| 地缘政治 | 俄乌/中东/台海重大升级（战争、制裁级别） | 全市场 + 港股 |
| 央行紧急行动 | 央行非例行加息/降准，超出市场预期 | 全市场 |
| 美股崩盘 | 纳指/标普单日跌幅 > 3% | A股 + QDII |
| A股流动性危机 | 沪深两市跌幅 > 5%，或涨跌停家数极端失衡 | 全市场 |
| 持仓黑天鹅 | 持仓对应板块突发重大监管打击/列入实体清单 | 对应持仓 |

**联动操作：**
```
触发🔴 →
  ① 跳过当日所有买入计划
  ② 如有对应持仓 → 检查是否触发卖出条件
     ├─ 已触发 → 立即执行卖出
     └─ 未触发 → 密切关注，次日9:15前复评
  ③ 在复盘报告中详细记录：
     ├─ 事件描述
     ├─ 当前持仓受影响程度
     └─ 次日操作预案（买/卖/观望）
```

---

### 🟠 中等（升级/降级）— 改变信号权重和持仓优先级

**判定条件（满足任一）：**

| 类型 | 触发条件 | 影响板块 | 加/减分 |
|------|---------|---------|:-------:|
| 降准/降息 | 央行宣布降准或降息（非例行） | 金融/地产/消费 | **加分** |
| 房地产重磅支持 | 国务院/住建部发布实质性松绑政策 | 地产链/银行/家电 | **加分** |
| 科技/芯片补贴 | 工信部/发改委发布实质性科技补贴或税收优惠 | 科创芯片/半导体/AI | **加分** |
| 新能源支持 | 新能源汽车购置税减免延续或加码 | 新能源/储能/电池 | **加分** |
| 医保谈判 | 医保谈判大幅压价某药品/耗材 | 医药/医疗器械 | **减分** |
| 美联储FOMC | 利率决议超预期鹰派/鸽派 | QDII（纳指/标普） | **减/加分** |
| 美非农/CPI | 数据显示通胀/就业超预期 | QDII | **减/加分** |
| 南向大资金 | 港股通单日净流入/流出 > 50亿 | 港股ETF | **加分** |
| 美科技股财报 | 英伟达/苹果/微软/谷歌业绩大幅miss | 纳指ETF | **减分** |
| 美科技股财报 | 英伟达/苹果/微软/谷歌业绩大幅beat | 纳指ETF | **加分** |
| 产业限制 | 某板块被列入出口管制/制裁名单 | 对应ETF | **减分** |

**联动操作（以板块加分为例，降分同理）：**

```
假设：工信部发布实质性半导体补贴政策
        ↓
① 识别受益ETF：512480(半导体)/588200(科创芯片)/588170(科创半导体)
        ↓
② 对这3只ETF信号升一档：
   ├─ ⏸ → ⭐（立即纳入观察，不再等零轴上金叉）
   ├─ ⭐ → ⭐⭐（次日优先买入）
   └─ ⭐⭐ → 维持（已是最强，不降分）
        ↓
③ 如现有持仓中有对应板块 → 优先保留，不因信号弱而降级
        ↓
④ 记录在复盘报告：加分标的 + 加分理由 + 次日操作
```

**具体加分规则（信号升降档）：**

| 原信号 | 重大利好(🟠) | 重大利空(🟠) |
|--------|:-----------:|:-----------:|
| ⭐⭐ | ⭐⭐（维持最强） | ⭐⭐（但MACD死叉则立即降⭐） |
| ⭐ | ⭐⭐（升级，提前买入窗口） | ⏸（降级，需重新等零轴上金叉） |
| ⏸ | ⭐（降一档，但不跳过买入条件） | ⏸（维持，观望） |

### 🟡 轻度（记录不干预）— 记录但不改决策

**判定标准：** 有一定重要性，但不足以改变当下买卖判断

| 例子 | 处置 |
|-----|------|
| 人大常委会一般性议题讨论 | 记录方向（利好/利空/中性），不改变信号 |
| 美联储官员非决议性讲话 | 记录偏鹰/偏鸽，次日关注期货反应 |
| 上市公司一般性公告 | 不处理（ETF投资者不关注个股） |
| 财经媒体一般性市场分析 | 不处理 |
| 外资小幅流出/流入（<20亿/日）| 不处理 |
| 盘后龙虎榜数据 | 记录大额买卖，不改变ETF决策 |

**操作：** 简单记录在复盘报告"盘后信息"一栏，**不改变任何信号评分和持仓决策**。

## 四、时间节点操作规范

### 21:00 复盘节点（主要）
```
① 收集当日A股/港股收盘后 ~ 21:00 的重要新闻
② 评估对持仓和候选的影响
③ 执行信号扫描
④ 如有持仓触发卖出 → 立即制定换股计划
⑤ 生成复盘报告
⑥ 次日09:15前执行换股计划
```

### 09:00 晨读节点（辅助）
```
① 查看隔夜美股收盘 + 期货表现（A50/纳指期货）
② 评估昨夜重大国际新闻
③ 调整次日操作计划（如有重大变化）
④ 如美股暴跌 > 3% → 当日跳过所有QDII买入
```

## 五、尺度自我约束（我的判断标准）

**以下是我在把握尺度时的内部规则：**

1. **官方 > 传闻**：只处理官方媒体或权威机构发布的新闻，自媒体/传闻不处理
2. **实质 > 口号**：政策类新闻，必须有具体数字/措施，空洞口号不处理
3. **超预期才处理**：加息/降息等，要超出市场预期才算🟠，符合预期不处理
4. **时效性**：重大新闻（🔴）→ 立刻处理；中等（🟠）→ 21:00复盘时处理
5. **不追涨杀跌**：因新闻大跌不恐慌卖出；因新闻大涨不追高买入
6. **ETF优于个股**：行业政策以ETF为主判断依据，不被单只个股误导

**信息来源优先级（用于评估消息真伪和重要性）：**

```
最高权威：国务院官网、央行官网、美联储官网
高权威：    新华社、人民日报、证监会官网、发改委官网
中权威：    财联社、证券时报、上证报、Wind
参考：      东方财富、雪球、36氪
外围参考：  Bloomberg、WSJ、路透社
```
