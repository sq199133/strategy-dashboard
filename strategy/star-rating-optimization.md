# 星级评级标准优化报告

## 优化背景

**原问题：**
1. 星级集中在低分段，⭐⭐⭐⭐⭐标的很少
2. 分数制（0-12分）不够直观，难以理解上涨概率
3. 相关性筛选与星级评分脱节

**用户需求：**
1. 最多5颗星（已满足）
2. 基于后续上涨概率、上升幅度预期来评级
3. 兼顾相关性分散

---

## 新评级模型（v3.1）

### 核心公式

```
综合评分 = 上涨概率 × 预期涨幅 × 100

星级 = f(综合评分)
```

---

### 一、上涨概率评估

**基准概率：** 50%（市场随机涨跌）

| 因子类别 | 因子 | 条件 | 概率贡献 |
|---------|------|------|:-------:|
| **趋势** | 多头排列 | MA20>MA50>MA200 | +25% |
| | 中期向上 | MA20>MA50 | +20% |
| | 短期向上 | 价格>MA20，MA20向上 | +15% |
| | 站上MA20 | 价格>MA20 | +10% |
| **动量** | 强势启动 | MACD零轴上金叉 | +25% |
| | 多头运行 | MACD零轴上 | +15% |
| | 反弹信号 | MACD金叉（零轴下） | +10% |
| | 动量加速 | MACD零轴上，红柱放大 | +20% |
| **强弱** | 强势领涨 | 20日涨幅>大盘5%+ | +20% |
| | 跑赢大盘 | 20日涨幅>大盘 | +15% |
| | 近期强势 | 5日涨幅>大盘 | +10% |
| **成交** | 放量启动 | 成交量>20日均量1.5倍 | +15% |
| | 量能配合 | 成交量>20日均量 | +10% |
| **政策** | 重大利好 | — | +10% |
| | 一般利好 | — | +5% |
| | 中性 | — | 0% |
| | 一般利空 | — | -10% |
| | 重大利空 | — | -20% |

**上涨概率 = 50% + Σ(因子概率贡献)**

**上限：** 75%（市场不确定性始终存在）

### 二、预期涨幅评估

| 因子类别 | 条件 | 涨幅贡献 |
|---------|------|:-------:|
| **突破** | 突破20日新高 | +5% |

| **MACD** | 零轴上金叉 | +8% |

**预期涨幅 = Σ(涨幅贡献)**

**范围：** 0-15%

| 星级 | 上涨概率 | 预期涨幅 | 综合评分 | 含义 |
|:----:|:-------:|:-------:|:--------:|------|
| ⭐⭐⭐⭐⭐ | ≥75% | ≥8% | ≥60 | 强烈买入 |

| ⭐ | <30% | <1% | <10 | 观望/卖出 |

### 四、相关性筛选规则

**同星级选择：**
1. 计算候选之间的相关性矩阵
2. 从最高分开始，贪心选择maxCorr≤0.70的组合
3. 每次选择后，排除与已选高相关的候选

**跨星级选择：**
- 星级优先于相关性
- 即使⭐⭐⭐⭐⭐相关性0.75，仍优先于⭐⭐⭐⭐
- 但最多持仓1只高相关(>0.70)标的

**同指数处理：**
- 只选规模最大、流动性最好的一只
- 其他同指数ETF降1星处理

---

### 五、星级动态调整

| 事件 | 调整 |
|------|------|
| 重大利好新闻 | +1星 |
| 重大利空新闻 | -1星 |
| 突破关键阻力 | +1星 |
| 跌破关键支撑 | -1星 |
| 与已持仓相关性>0.80 | -1星 |

## 实战示例

### 案例：创业板50ETF（159681）

**因子分析：**

| 类别 | 条件 | 概率贡献 | 涨幅贡献 |
|------|------|:-------:|:-------:|
| 趋势 | 价格>MA20>MA50 | +20% | — |
| 动量 | MACD零轴上运行 | +15% | +5% |
| 强弱 | 20日涨幅>大盘 | +15% | — |
| 成交 | 放量1.5倍 | +15% | — |
| 突破 | 创20日新高 | — | +5% |

**计算：**
- 上涨概率 = 50% + 65% = 115% → 上限75%
- 预期涨幅 = 10%
- 综合评分 = 75% × 10% × 100 = **75分**

**结果：** ⭐⭐⭐⭐⭐
