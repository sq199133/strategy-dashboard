## 一、策略概述

**核心理念**: 只做多头趋势中的MACD金叉共振，在趋势回调结束后精准切入，享受主升浪。

**底层逻辑**:
- 20日均线(MA20) 识别中长期趋势方向
- MACD 判断短期动量变化与最佳买入时机
- 只有在上升趋势中，MACD金叉的成功率才高

---

## 二、买入条件 (BUY)

### 基础条件（必须同时满足）

| 条件 | 说明 | 判断标准 |
|------|------|----------|
| ① 价格站上MA20 | 确认处于多头趋势 | 收盘价 > MA20 |
| ② MA20向上 | 趋势向上 | MA20_today > MA20_yesterday |
| ③ MACD零轴上金叉 | 最佳买入时机 | DIF从下向上穿越DEA，且两者 > 0 |

### 增强条件（满足越多越好）

| 条件 | 加分 | 说明 |
|------|:----:|------|
| MACD红柱放大 | +2 | 动量持续增强 |
| 成交量放大 | +1 | 资金流入确认 |
| 突破前高 | +2 | 打开上涨空间 |
| 站在涨停板上 | +3 | 强势信号 |

### 排除条件（一票否决）

- ❌ MACD零轴下金叉（弱势反弹，可能失败）
- ❌ 放量跌破MA20（趋势可能反转）
- ❌ 大盘处于MA20之下（系统性风险）

---

## 三、卖出条件 (SELL)

### 主要卖出信号（满足任一即卖出）

| 优先级 | 条件 | 说明 |
|:------:|------|------|
| 1 | 价格跌破MA20 | 趋势转弱 |
| 2 | MACD死叉 | 动量反转 |
| 3 | MACD红柱转绿柱 | 短期调整确认 |
| 4 | 亏损超过5% | 止损保护 |
| 5 | 涨幅超过30% | 止盈部分 |

### 卖出规则细化

根据回测数据，以下卖出规则效果最优：

| 卖出方式 | 年化收益 | 最大回撤 | 交易次数 |
|----------|:--------:|:--------:|:--------:|
| MACD死叉 | **6.8%** | **8.8%** | 16次 |
| MACD红柱转绿 | 6.8% | 8.8% | 16次 |
| MA20跌破 | 6.0% | 11.4% | 15次 |

**结论**: 优先使用 MACD死叉 作为主要卖出信号

## 四、五星评分系统

### 评分规则（满分12分）

```
基础分（4分）:
  - 站上MA20 → 1分
  - MA20向上 → 1分  
  - MACD零轴上 → 1分
  - MACD金叉 → 1分

动量分（4分）:
  - MACD红柱 > 0 → 1分
  - 红柱连续放大 → 1分
  - 5日涨幅 > 3% → 1分
  - 20日涨幅 > 10% → 1分

趋势分（4分）:
  - 跑赢创业板指 → 2分
  - 零轴上金叉 → 2分
```

## 五、选股池规则

### ETF选择原则

1. **同指数只选一只**: 避免重复持仓
2. **相关性过滤**: maxCorr ≤ 0.70
3. **流动性优先**: 规模 > 10亿优先
4. **全市场覆盖**: A股 + 港股 + 美股 + 商品

### 当前标的池

- **A股宽基**: 17只 (沪深300/中证500/创业板/科创50等)
- **科技**: 14只 (芯片/通信/AI/半导体等)
- **高端制造**: 10只 (机器人/军工/卫星等)
- **新能源**: 11只 (光伏/电池/新能源车等)
- **消费医药**: 11只 (医药/消费/食品等)
- **金融地产**: 4只 (银行/证券/地产)
- **周期资源**: 8只 (有色/稀土/稀有金属)
- **策略指数**: 6只 (成长/红利/央企等)
- **跨境QDII**: 30只 (纳指/恒生/日经/德国等)
- **商品**: 9只 (黄金/豆粕/能源化工)

**合计**: 120只ETF

## 九、操作流程

### 每日复盘 (18:00)

```
1. 获取全市场ETF行情数据
2. 计算MA20和MACD指标
3. 按五星评分系统打分
4. 筛选 BUY 信号 (≥3星)
5. 检查持仓是否触发卖出
```

## 十、策略局限

1. **数据限制**: 腾讯API仅提供150条K线，历史覆盖有限
2. **市场环境**: 震荡市中可能频繁交易
3. **流动性风险**: 小规模ETF可能有流动性问题
4. **系统性风险**: 大盘暴跌时策略可能失效
