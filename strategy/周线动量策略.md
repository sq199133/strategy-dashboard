## 一、策略核心

| 组件 | 作用 |
|------|------|
| **2周动量** | 衡量短期价格趋势强度（当前价 / 2周前价 - 1） |
| **MA8/MA21** | 判断趋势方向（8周均线在21周均线之上=多头） |
| **偏离度≤15%** | 防追高过滤（价格偏离MA21不超过15%） |
| **动量差距阈值3%** | 换仓门槛（候选动量至少比最弱持仓高3%才值得换） |
| **止损机制** | 成本止损-8% + 高点回撤止损-10% |
| **集中持仓** | 最多3只，均分配置，火力集中 |

## 二、买入条件（同时满足三条件）

| # | 条件 | 计算方式 | 说明 |
|:--:|------|---------|------|
| 1 | **2周动量 > 0** | (当前周收盘价 / 2周前收盘价) - 1 > 0 | 价格处于上升趋势 |
| 2 | **价格 > MA8 > MA21** | MA8=8周均线，MA21=21周均线 | 短期和中期趋势均向上，趋势共振 |
| 3 | **偏离度 ≤ 15%** | (当前价 / MA21) - 1 ≤ 15% | 防止追高，偏离过大时趋势可能逆转 |

### 指标计算说明

- **周K线构建**：日K线按ISO周聚合，每周五收盘价为周收盘价
- **MA8**：最近8周收盘价的算术平均（约2个月，比MA3更稳定）
- **MA21**：最近21周收盘价的算术平均（约5个月，中期趋势基准）
- **2周动量**：当前周收盘价 / 2周前周收盘价 - 1
- **偏离度**：当前周收盘价 / MA21 - 1
- **数据源**：腾讯API前复权日K线（`kline_dayqfq`），周线聚合后使用

### 信号时序

1. **周五收盘**：计算所有ETF的动量、MA、偏离度
2. **生成信号**：按动量从高到低排序，取前3只
3. **周一开盘**：执行买入/卖出

> ⚠️ 信号生成与执行不在同一天，避免前视偏差

## 三、换仓规则

### 3.1 三种退出机制

#### ① 强制退出（必须卖）
持仓ETF**不再满足入场三条件**中任一条时卖出：
- 动量转负（2周涨幅为负）
- 趋势破坏（价格 ≤ MA8，或 MA8 ≤ MA21）
- 偏离度超15%（涨过头，趋势过热）

→ 说明持仓已不再强势，必须换掉。

#### ② 止损退出（必须卖）
- **成本止损：** 当前价 ≤ 买入价 × 92%（亏损8%就砍）
- **高点回撤止损：** 当前价 ≤ 持有期间最高价 × 90%（从最高点回撤10%锁利）

→ 每日收盘价检查，任一先触发则次日开盘卖出。

#### ③ 主动换仓（选择性卖）
持仓仍然合格，但出现了更好的标的：
- 候选池某只ETF动量比持仓中最弱的那只**高出3%以上**
- 且持仓不是当周刚买的

→ 卖掉最弱的，换成更强的。3%门槛过滤掉不值得的微弱优势。

### 3.2 换仓判断流程

```
每周五收盘后：
1. 计算所有ETF的动量/MA8/MA21/偏离度，筛选合格标的
2. 按动量排序，取Top3作为候选池
3. 对比当前持仓：

   【强制退出】持仓不满足入场条件 → 周一卖出
   【止损退出】成本亏8%或高点回撤10% → 次日卖出
   【主动换仓】持仓仍合格，候选比最弱持仓高3%+ → 周一卖出最弱、买入最强
   【继续持有】持仓合格且候选优势不足3% → 不动

4. 无合格标的 → 空仓持有现金
```

### 3.3 换仓成本

| 成本项 | 估算 |
|--------|------|
| 卖出佣金 | ~0.05% |
| 买入佣金 | ~0.05% |
| 冲击成本/滑点 | ~0.1-0.2% |
| 单次换仓总成本 | ~0.2-0.3% |

> 3%差距阈值 → 净收益约2.7%，足以覆盖换仓成本

### 3.4 动量差距阈值回测依据

回测323周，3组参数×10个差距阈值，结论：
- 1-2%理论最优，但考虑滑点+执行延迟，3-5%更实用
- 完全不换仓年化掉30-50%，证明适度轮动有价值
- 323周里主动换仓仅40-50次（约每6-7周1次），换仓频率本身就低
- 差距5%以上效果反而差——等大差距时趋势已走完，换进去容易追高

## 四、持仓管理

### 资金配置

- **总资金**：5万元
- **持仓数量**：最多3只
- **配置方式**：等权配置，每只约1.67万元，买100股整数倍
- **空仓**：无合格标的时空仓持有现金（回测空仓率约5-8%）

### 换仓频率

- **检查频率**：每周一检查，根据上周五信号执行
- **实际换仓频率**：约每6-7周1次主动换仓（323周回测数据）
- **强制退出**：持仓跌出候选时立即退出，不受差距阈值限制

## 五、止损规则

| 类型 | 条件 | 执行 |
|------|------|------|
| **成本止损** | 亏损 ≥ 8%（相对买入价） | 下一个交易日开盘卖出 |
| **高点回撤止损** | 从持仓最高点回撤 ≥ 10% | 下一个交易日开盘卖出 |
| **偏离度超限退出** | 偏离度 > 15%（趋势过热） | 周一开盘卖出 |

### 止损说明

- 成本止损和高点回撤止损以每日收盘价检查，触发后次日开盘执行
- 偏离度超限退出为周线级别信号，周五计算、周一执行
- 成本止损保护本金，高点回撤止损锁定利润
- 止损后资金回到现金池，下周一重新参与信号筛选

## 六、标的池

- **版本**：91只ETF（2026-05-02修复）
- **文件**：`D:\QClaw_Trading\data\etf_pool.js`
- **历史数据**：`D:\QClaw_Trading\data\history_long\`（JSON格式，前复权）
- **数据源**：腾讯API `kline_dayqfq`（日K线前复权），周线聚合后使用

### 筛选逻辑

1. 从91只ETF中计算每只的2周动量
2. 过滤：动量>0 且 价格>MA8>MA21 且 偏离度≤15%
3. 同一指数类别只选动量最高的1只（去重）
4. 按动量从高到低排序，取前3只
5. 对比持仓，按换仓规则（第三节）决定是否调仓

### 关于前复权数据

本策略使用**前复权**（qfq）数据计算所有指标和信号。前复权将历史价格按分红除权向下调整，使收益计算包含分红再投资收益，反映真实总回报。这是量化策略的标准做法。

**注意：** 买入卖出时使用实际市场价格（非复权价），信号计算用前复权价，两者存在微小差异（分红导致的调整），但对ETF影响极小（ETF分红频率低、金额小）。

## 七、风险提示

1. **MA21滞后风险**：21周均线约5个月，下跌初期可能仍显示"趋势向上"，需止损机制补充
2. **3只集中风险**：单一标的波动对组合影响大（每只占33%），极端行情下3只可能同时亏损
3. **偏离度15%仍可能追高**：偏离15%时买入，若MA21未跟上，回撤空间约13%
4. **2022年回撤-24.8%**：熊市中策略无法避免亏损，但在72组参数中已是最优平衡
5. **回测局限**：早期数据覆盖不足（2020年仅部分ETF有数据），后期代表性更强
6. **前复权与实盘差异**：信号基于前复权价计算，实盘以市价成交，分红日附近可能有小幅偏差
7. **MA8/21参数稳定性**：7年回测中6年正收益，但未来市场结构变化可能导致参数失效

## 八、操作流程

### 每周五收盘后
1. 运行筛选脚本，计算全部91只ETF的动量/MA8/MA21/偏离度
2. 生成合格标的列表，按动量排序取Top3
3. 对比当前持仓：
   - 检查强制退出条件（动量/MA/偏离度）
   - 检查止损条件（成本-8%或高点回撤-10%）
   - 检查主动换仓条件（动量差距≥3%）
4. 生成操作指令：卖出/买入/持有

### 每周一开盘
1. 执行卖出（强制退出、止损或主动换仓）
2. 等权买入新的Top3标的
3. 记录交易到持仓跟踪文件

### 每日收盘
1. 检查止损条件（成本-8%或高点回撤-10%）
2. 触发止损则次日开盘卖出
