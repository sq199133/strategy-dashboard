## 一、策略核心

| 组件 | 作用 |
|------|------|
| **LB周动量** | 衡量中期价格趋势强度（当前价 / LB周前价 - 1） |
| **MA5/MA21** | 判断趋势方向（5周均线在21周均线之上=多头） |
| **偏离度≤10%** | 防追高过滤（价格偏离MA21不超过10%） |
| **G3过滤** | 标的近三周涨幅≥0% 且 近一周涨幅≥-1% |
| **2只均分持仓** | 集中持有最强2只ETF |
| **止损机制** | 成本止损-8% + 高点回撤止损-10% |

## 二、买卖规则

### 买入规则

每周五收盘后执行以下步骤：

1. **计算动量**：LB=3周动量 = 本周收盘 / 3周前收盘 - 1
2. **趋势过滤**：MA5 > MA21（5周均线在21周均线之上）
3. **偏离度过滤**：价格偏离MA21 ≤ 10%
4. **G3过滤**：近三周涨幅≥0% 且 近一周涨幅≥-1%
5. **排序**：通过全部过滤的ETF按LB动量从高到低排序
6. **买入**：买入前2名（均分资金）

### 卖出规则

每周五检查持仓ETF：

1. **不在合格列表** → 卖出
2. **动量≤0** → 卖出
3. **MA5 ≤ MA21** → 卖出
4. **偏离度 > 10%** → 卖出
5. **G3过滤不通过** → 卖出
6. **止损触发** → 卖出（成本-8% 或 高点回撤-10%）

## 三、参数说明

| 参数 | 值 | 说明 |
|------|------|------|
| `ma_s` | 5 | 短期均线（周） |
| `ma_l` | 21 | 长期均线（周） |
| `lb` | 3 | 回看周期（周） |
| `max_dev` | 10% | 最大偏离度 |
| `top_n` | 2 | 持仓数量 |
| `hs300_threshold` | -100 | 市场过滤阈值（禁用） |

## 四、风险提示

1. **2010-2015年基本无效** - 策略在ETF市场不成熟时期表现差
2. **过拟合风险** - 参数基于历史数据优化，未来可能失效
3. **2026年回撤** - 当前（2026-W23）已回撤-9.8%，最大回撤-20.9%
4. **持仓集中** - 仅持2只ETF，单只ETF风险较高
