> **策略类型：** 周线级别动量轮动
> **回测验证：** 2010-W01~2026-W24，190只ETF，795周（数据源：腾讯API前复权）

## 一、策略核心

| 组件 | 参数 | 说明 |
|------|------|------|
| **多周期动量评分** | w1=40%, w3=40%, w8=20% | 合成本周/3周/8周三周期动量 |
| **MA5/MA21** | 短期5周, 长期21周 | 价格>MA5>MA21 趋势过滤 |
| **偏离度上限** | ≤15% | 价格偏离MA21不超过15% |
| ~~G3过滤~~ | ~~3周≥0% 且 1周≥-1%~~ | ~~已移除：降年化1.6%，2026年更差~~ |
| **ATR波动率过滤** | ATR14/ATR21 ≥ 0.85 | 波动率严重收缩时跳过（动量多为噪声） |
| **持仓数量** | 3只均分 | 集中持有最强3只ETF |
| **止损机制** | 成本-8% 或 高点-10% | 双重止损 |

### 评分公式

```
SCORE = w1 × MOM1W + w3 × MOM3W + w8 × MOM8W
       = 0.4 × MOM1W + 0.4 × MOM3W + 0.2 × MOM8W
```

- MOM1W = 本周收盘 / 1周前收盘 - 1
- MOM3W = 本周收盘 / 3周前收盘 - 1
- MOM8W = 本周收盘 / 8周前收盘 - 1

**设计理念：** 纯3周动量排序在震荡市容易买在反转点。引入1周动量捕捉刚启动的趋势，8周动量做长周期质量校验，减少追高买入在反转点的概率。ATR波动率过滤：当ATR(14)/ATR(21) < 0.85时，说明近期波动率严重收缩，此时动量信号多为噪声，跳过该标的。

## 三、扫描流程

```
每周五收盘 →
  1. 获取195只ETF周线数据（腾讯API）
  2. 计算各只 MOM1W, MOM3W, MOM8W
  3. 计算复合评分 SCORE = 0.4×MOM1W + 0.4×MOM3W + 0.2×MOM8W
  4. 计算ATR比率 = ATR(14周)/ATR(21周) — 若<0.85则跳过
  5. 过滤：
    a) MA5 > MA21（上升趋势）
    b) 收盘价 > MA5（短期强势）
    c) 偏离度 ≤ 15%（防追高）
    d) G3: MOM3W ≥ 0% 且 MOM1W ≥ -1%
    e) ATR比率 ≥ 0.85 或 无ATR数据（放行）
  6. 按评分排序，同类别去重，取前3只等权持仓
  7. 对比当前持仓，生成买卖指令
  8. 止损检查：成本-8% 或 高点-10%
```

### 卖出条件（任一条即卖出）

1. 不在本周扫描合格列表
2. MOM1W ≤ 0%（动量衰竭）
3. MA5 ≤ MA21（趋势破坏）
4. 偏离度 > 15%（严重偏离）
5. G3过滤不通过
6. ATR比率 < 0.85（波动率异常收缩）
7. 止损触发（成本-8% 或 高点-10%）
