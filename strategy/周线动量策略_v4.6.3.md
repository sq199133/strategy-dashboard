# 周线动量策略 v4.6.3

> **版本：** v4.6.3（2026-07-05 更新，移除赛道去重）
> **策略类型：** 周线级别动量轮动 + 技术形态加分 + 量能过滤
> **回测验证：** IS(2017~2022) + OOS(2023~2026)，195只ETF
> **数据源：** 腾讯API前复权周线（history_long_v2/）

---

## 一、策略核心

| 组件 | 参数 | 说明 |
|------|------|------|
| **多周期动量评分** | w1=40%, w3=40%, w8=20% | 合成本周/3周/8周三周期动量 |
| **形态加分** | C+0.02, B1=0 | 仙人指路加分（红三兵已移除） |
| **高量能过滤** | 量比>1.5跳过 | 避开放量出货信号（v4.6.2新增） |
| **MA5/MA21** | 短期5周, 长期21周 | 价格>MA5>MA21 趋势过滤 |
| **偏离度上限** | ≤15% | 价格偏离MA21不超过15% |
| **ATR波动率过滤** | ATR14/ATR21 ≥ 0.85 | 波动率严重收缩时跳过 |
| **持仓数量** | 3只均分 | 集中持有最强3只ETF |
| **止损机制** | 成本-8% 或 高点-10% | 双重止损 |

### 评分公式

```
基础分 = 0.4 × MOM1W + 0.4 × MOM3W + 0.2 × MOM8W
仙人指路(C)触发 → +0.02
高量能(量比>1.5) → 跳过（不进入候选）
最终分 = 基础分 + C加分
```

**注：** v4.6.3移除了赛道去重逻辑（同赛道只留第一名），允许同赛道多只ETF入选。

## 二、形态加分详解

### C型：仙人指路（+0.02）

**识别条件（需同时满足）：**

| 条件 | 阈值 | 含义 |
|------|------|------|
| 阳线 | close > open | 本周收阳 |
| 上影长度 | 上影 > 实体 | 上影线长度超过实体本身 |
| 下影长度 | 下影 < 实体×50% | 下影线短，不砸漏 |
| 成交量 | 量比 < 1.5×10周均量 | 温和放量，非天量 |
| 均线状态 | close > MA5 > MA21 | 均线多头排列 |
| 累计涨幅 | 20周涨幅 < 50% | 低位/中段，非高位追涨 |

**原理：** 长上影+阳线+下影短 = 主力试盘后洗盘，拉升前兆。叠加均线多头说明在上升趋势中，是有效的动量加速信号。

## 三、高量能过滤（v4.6.2新增）

### 量比定义

```
量比 = 本周成交量 / 10周成交量均值
```

### 过滤逻辑

**跳过条件：** 量比 > 1.5

**原理：**
- 量比>1.5×10周均量通常是**高位放量出货**信号
- 动量策略在此时买入往往会买在成交密集区的顶部
- 跳过这些候选后，只买入正常量能的ETF，买入成本更优

### 回测验证（IS 2017~2022）

| 配置 | 年化 | 最大DD | 夏普 | d年化 | d夏普 |
|------|------|--------|------|-------|-------|
| 基线(无过滤) | +16.86% | -25.2% | 0.907 | — | — |
| **跳过 量比>1.5** | **+19.10%** | **-16.8%** | **1.038** | **+2.24%** | **+0.131** |

**结论：** 高量能过滤在IS期显著提升年化收益（+2.24%）并降低最大回撤（-8.4%），夏普比率从0.907提升至1.038。

### OOS验证（2023~2026）

| 配置 | 年化 | 最大DD | 夏普 |
|------|------|--------|------|
| 基线(无过滤) | +28.39% | -27.4% | 1.030 |
| 跳过 量比>1.5 | +26.63% | -28.9% | 1.052 |

**结论：** OOS期夏普更高（1.052 vs 1.030），但年化略低（-1.76%）。OOS期（2023~2026）是极端ETF牛市，不足以区分因子优劣。需要更长熊市样本验证回撤保护能力。

## 四、买入规则

1. **候选筛选：**
   - MA5 > MA21（趋势过滤）
   - 偏离度 ≤ 15%（不过热）
   - ATR比率 ≥ 0.85（波动率正常）
   - **量比 ≤ 1.5（v4.6.2新增，跳过高位放量）**
   
2. **评分排序：**
   - 计算复合动量评分（SC40-40-20）
   - 仙人指路(C)触发 +0.02
   - 按调整后评分降序排列
   
3. **买入执行：**
   - 每周五收盘后扫描，下周一开盘调仓
   - 最多持有3只，等权重分配

---

## 五、卖出规则

触发任一条件即卖出：

| 条件 | 阈值 | 说明 |
|------|------|------|
| 动量转负 | MOM3W ≤ 0 | 3周动量转负，趋势破坏 |
| 趋势破坏 | MA5 ≤ MA21 | 短期均线下穿长期均线 |
| 偏离度过高 | 偏离度 > 15% | 价格远离均线，回调风险 |
| 止损（成本） | 现价 ≤ 买入价×92% | 亏损8%止损 |
| 止损（回撤） | 现价 ≤ 高点×90% | 从高点点回撤10%止损 |
