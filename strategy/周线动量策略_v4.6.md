# 周线动量策略 v4.6.1

> **版本：** v4.6.1（2026-07-03 更新，移除红三兵加分，C加分降至0.02）
> **策略类型：** 周线级别动量轮动 + 技术形态加分
> **回测验证：** 2014-W52~2026-W26，195只ETF，641周（数据源：腾讯API前复权）

---

## 一、策略核心

| 组件 | 参数 | 说明 |
|------|------|------|
| **多周期动量评分** | w1=40%, w3=40%, w8=20% | 合成本周/3周/8周三周期动量 |
| **形态加分** | C+0.02, B1=0 | 仙人指路加分（红三兵已移除） |
| **MA5/MA21** | 短期5周, 长期21周 | 价格>MA5>MA21 趋势过滤 |
| **偏离度上限** | ≤15% | 价格偏离MA21不超过15% |
| **ATR波动率过滤** | ATR14/ATR21 ≥ 0.85 | 波动率严重收缩时跳过 |
| **持仓数量** | 3只均分 | 集中持有最强3只ETF |
| **止损机制** | 成本-8% 或 高点-10% | 双重止损 |

### 评分公式

```
基础分 = 0.4 × MOM1W + 0.4 × MOM3W + 0.2 × MOM8W
仙人指路(C)触发 → +0.02
红三兵(B1)  → 已移除（回测显示负效果）
最终分 = 基础分 + C加分
```

## 二、形态加分详解

### C型：仙人指路（+0.02）

**识别条件（需同时满足）：**

| 条件 | 阈值 | 含义 |
|------|------|------|
| 阳线 | close > open | 本周收阳 |
| 上影长度 | 上影 > 实体 | 上影线长度超过实体本身 |
| 下影长度 | 下影 < 实体×50% | 下影线短，不砸漏 |
| 成交量 | 量比 < 1.5×10周均量 | 温和放量，非天量 |
| 均线状态 | close > MA5 > MA21 | 均线多头排列 |
| 累计涨幅 | 20周涨幅 < 50% | 低位/中段，非高位追涨 |

**原理：** 长上影+阳线+下影短 = 主力试盘后洗盘，拉升前兆。叠加均线多头说明在上升趋势中，是有效的动量加速信号。

### B1型：红三兵（已移除）

**回测结论（2026-07-03）：** B1+0.03导致年化损失-4.2%（+9.48% vs 基线+13.68%），在震荡市/熊市（2018/2022年）过度轮仓造成亏损累积。红三兵在熊市反弹中频繁触发假突破，已从策略中移除。

## 三、回测结果（2010-W01~2026-W26）

### 总体指标对比

| 指标 | v4.5.1（基线） | **v4.6.1（C+0.02）** | 变化 |
|:---|:---:|:---:|:---:|
| **年化收益** | +13.68% | **+14.40%** | +0.72% |
| **最大回撤** | -40.8% | **-40.8%** | 持平 |
| **夏普比率** | 0.606 | **0.636** | +0.030 |
| **胜率** | 54.1% | 53.4% | -0.7% |

> 注：C+0.02是回测16种配置中的最优解。
> **红三兵（B1）加分已移除**：B1+0.03导致年化降至+9.48%（损失-4.2%），在熊市震荡市过度轮仓。

### 核心效果

- **C+0.02最优**：年化+14.40%，夏普0.636，全面优于基线
- **C≥0.08有害**：年化降至+9.29%，开始错误提升弱势标的排名
- **B1红三兵负效果**：B1+0.03年化+9.48%，2022年熊市仅+0.8% vs 基线+14.2%（差距15%）
- **推荐参数：C+0.02, B1=0.00**

## 四、扫描流程

```
每周五收盘 →
  1. 获取195只ETF周线数据（腾讯API，本地文件回退）
  2. 计算各只 MOM1W, MOM3W, MOM8W
  3. 计算基础评分 = 0.4×MOM1W + 0.4×MOM3W + 0.2×MOM8W
  4. 计算ATR比率 = ATR(14周)/ATR(21周) — 若<0.85则跳过
  5. 形态识别：
     a) C型（仙人指路）：阳线+长上影+短下影+温和量+均线多头+低位
     b) B1型（红三兵）：连续3周阳线+低点抬高+量能稳定
  6. 最终分 = 基础分 + C×0.05 + B1×0.03
  7. 过滤：
     a) MA5 > MA21（上升趋势）
     b) 收盘价 > MA5（短期强势）
     c) 偏离度 ≤ 15%（防追高）
     d) ATR比率 ≥ 0.85（波动率正常）
  8. 按最终分排序，同类别去重，取前3只等权持仓
  9. 对比当前持仓，生成买卖指令
  10. 止损检查：成本-8% 或 高点-10%
```

### 卖出条件（任一条即卖出）

1. 不在本周扫描合格列表（按最终分排序后）
2. MOM1W ≤ 0%（动量衰竭）
3. MA5 ≤ MA21（趋势破坏）
4. 偏离度 > 15%（严重偏离）
5. ATR比率 < 0.85（波动率异常收缩）
6. 止损触发（成本-8% 或 高点-10%）

## 七、回测脚本说明

### 形态回测相关脚本

| 脚本 | 用途 |
|------|------|
| `uppershadow_backtest.py` | 上影线分类回测（原始54,472周次） |
| `uppershadow_qualified.py` | 限定策略合格ETF的上影线回测（13,014次） |
| `shadow_classified_backtest.py` | 精细上影线分类（A1/A2/B1/B2/C型） |
| `shadow_strategic_backtest.py` | 形态叠加策略回测 |

| `backtest_pattern_yearly.py` | 形态加分组年度验证 |

### 关键回测结论

- **A1型（高位放量长上影）**：策略合格ETF中从未触发（风控自动规避）
- **B2型（均线附近缩量小上影）**：减分反而更差，原因：牛市有效但在震荡市过度轮仓
- **C型（仙人指路）**：加分有效，+0.05时效果最优
- **B1型（红三兵）**：最强因子，同时提升年化和降低回撤
- **空头形态**：单独使用无超额收益，不加入卖出规则
