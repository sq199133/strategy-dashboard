## 二、策略核心

| 组件 | 参数 | 说明 |
|------|------|------|
| **多周期动量评分** | w1=50%, w3=50%, w8=0% | 合成本周/3周双周期动量（8周无效） |
| **形态加分** | C+0.02, B1=0 | 仙人指路加分（红三兵已移除） |
| **高量能过滤** | 量比>1.5跳过 | 避开放量出货信号（v4.6.2） |
| **MA5/MA21** | 短期5周, 长期21周 | 价格>MA5>MA21 趋势过滤 |
| **偏离度上限** | **≤20%** | 价格偏离MA21不超过20%（v4.7终版优化） |
| **ATR波动率过滤** | ATR14/ATR21 ≥ 0.85 | 波动率严重收缩时跳过 |
| **持仓数量** | **1只（top=1）** | 集中持有最强1只ETF |
| **赛道去重** | **已还原** | 同赛道只留第一名 |
| **止损机制** | 成本-8% 或 高点-10% | 双重止损 |

### 评分公式

```
基础分 = 0.50 × MOM1W + 0.50 × MOM3W
仙人指路(C)触发 → +0.02
高量能(量比>1.5) → 跳过（不进入候选）
赛道去重 → 同赛道只保留最高分
最终分 = 基础分 + C加分
```

**注：** 8周动量权重优化为0（网格搜索最优），说明短期动量比中期动量更有效。

## 三、赛道去重规则（v4.7 还原）

**【规则】** 评分排序后，按赛道类别去重：同一赛道（category字段）只保留评分最高的ETF。

**作用：**
- 防止同赛道ETF同时入选（如多只机器人ETF）
- 提升分散度，降低系统性回撤
- 根据回测验证：有去重回撤更低（-19.7% vs -26.5%）
