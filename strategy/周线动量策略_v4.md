## 一、策略核心

| 组件 | 作用 |
|------|------|
| **3周动量** | 衡量短期价格趋势强度（当前价 / 3周前价 - 1） |
| **MA5/MA21** | 判断趋势方向（5周均线在21周均线之上=多头） |
| **偏离度≤15%** | 防追高过滤（价格偏离MA21不超过15%） |
| **G3过滤** | 标的近三周涨幅≥0% 且 近一周涨幅≥-1% |
| **3只均分持仓** | 分散风险同时保持集中 |
| **止损机制** | 成本止损-8% + 高点回撤止损-10% |

## 二、买入条件（同时满足四条件）

| # | 条件 | 计算方式 | 说明 |
|:--:|------|---------|------|
| 1 | **3周动量 > 0** | (当前周收盘价 / 3周前收盘价) - 1 > 0 | 价格处于上升趋势 |
| 2 | **价格 > MA5 > MA21** | MA5=5周均线，MA21=21周均线 | 短期和中期趋势均向上 |
| 3 | **偏离度 ≤ 15%** | (当前价 / MA21) - 1 ≤ 15% | 防止追高 |
| 4 | **G3过滤** | 近三周涨幅≥0% 且 近一周涨幅≥-1% | 过滤弱势标的 |

### 指标计算说明

- **周K线构建**：日K线按ISO周聚合，每周五收盘价为周收盘价
- **MA5**：最近5周收盘价的算术平均（约1个月，快速趋势）
- **MA21**：最近21周收盘价的算术平均（约5个月，中期趋势基准）
- **3周动量**：当前周收盘价 / 3周前周收盘价 - 1
- **偏离度**：当前周收盘价 / MA21 - 1
- **G3过滤**：
  - 近三周涨幅 = 当前周收盘价 / 3周前周收盘价 - 1 ≥ 0%
  - 近一周涨幅 = 当前周收盘价 / 上周收盘价 - 1 ≥ -1%
- **数据源**：腾讯API前复权日K线，周线聚合后使用

### 信号时序（严格无前视偏差）

1. **周五收盘**：计算所有ETF的动量、MA、偏离度、G3过滤
2. **生成信号**：按动量从高到低排序，取前3只
3. **下周一开盘**：执行买入/卖出（信号与执行差一周）

## 三、换仓规则

### 3.1 退出机制

#### ① 强制退出（必须卖）
持仓ETF**不再满足入场四条件**中任一条时卖出：
- 动量转负（3周涨幅为负）
- 趋势破坏（价格 ≤ MA5，或 MA5 ≤ MA21）
- 偏离度超限（>15%）
- G3过滤不通过（近三周涨幅<0% 或 近一周涨幅<-1%）

#### ② 止损退出（必须卖）
- **成本止损：** 当前价 ≤ 买入价 × 92%（亏损8%就砍）
- **高点回撤止损：** 当前价 ≤ 持有期间最高价 × 90%（从最高点回撤10%锁利）

### 3.2 换仓判断流程

```
每周五收盘后：
1. 计算所有195只ETF的动量/MA5/MA21/偏离度/G3过滤，筛选合格标的
2. 同一指数类别只选动量最高的1只（去重）
3. 按动量排序，取Top3作为候选
4. 对比当前持仓：
   【强制退出】持仓不满足入场条件 → 下周一卖出
   【止损退出】成本亏8%或高点回撤10% → 次日卖出
   【换仓】候选中有更优标的 → 下周一调整
   【继续持有】持仓合格且排名靠前 → 不动
5. 无合格标的 → 空仓持有现金
```

## 四、持仓管理

### 资金配置

- **总资金**：5万元
- **持仓数量**：最多3只
- **配置方式**：等权配置，买100股整数倍

### 换仓频率

- **检查频率**：每周一检查，根据上周五信号执行
- **G3过滤**：自动过滤弱势标的，减少无效换仓

## 五、止损规则

| 类型 | 条件 | 执行 |
|------|------|------|
| **成本止损** | 亏损 ≥ 8%（相对买入价） | 下一个交易日开盘卖出 |
| **高点回撤止损** | 从持仓最高点回撤 ≥ 10% | 下一个交易日开盘卖出 |
| **G3过滤不通过** | 近三周涨幅<0% 或 近一周涨幅<-1% | 下周一开盘卖出 |

## 七、风险提示

1. **G3过滤可能过度拟合**：三周≥0% + 本周≥-1%是基于2020-2026数据选出的最优参数，未来可能失效
2. **MA5敏感风险**：5周均线约1个月，信号切换比MA8更频繁
3. **LB=3过拟合风险**：回看3周是唯一有效参数（LB=4夏普从1.16暴跌至0.48）
4. **195只池覆盖不足**：部分ETF历史数据仅1-2年，早期回测代表性有限
