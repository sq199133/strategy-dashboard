# 周线动量策略 v5.1（集中版 + G3过滤）

> **版本：** v5.1（2026-06-06，基于v5.0加入G3过滤，跳过2024-W01/2025-W01异常数据）
> **策略类型：** 周线级别动量轮动（集中持仓版）
> **回测验证：** 2020-W01 ~ 2026-W18，195只标的，320周，T周信号→T+1周执行

---

## 一、策略定位

v4.0稳健版（MA5/21 D15 H5）年化18.7%、夏普0.85，虽然稳定但收益偏保守。
v5.0在同样的MA5/21 D15基础上，将持仓从5只缩减到3只，理由：

**持仓越集中，动量最强的标的权重越高。** 等权配置下排名4-5的标的动量显著弱于Top3，
缩减到3只等于自动淘汰最弱持仓，集中火力在最强动量上。

| 对比 | v5.0（无G3） | **v5.1（G3过滤）** | 变化 |
|------|------------|-------------------|------|
| 持仓数 | 3只 | **3只** | 不变 |
| G3过滤 | 无 | **三周≥0% + 本周≥-1%** | 新增 |
| 年化收益 | +25.8% | **+29.7%** | +3.9% |
| 最大回撤 | -25.1% | **-22.1%** | +3.0% |
| 夏普比率 | 0.97 | **1.31** | +35% |
| Calmar | 1.03 | **1.34** | +30% |

## 二、策略核心参数

| 参数 | 值 | 含义 |
|------|-----|------|
| **MA5/MA21** | 5周 / 21周 | 短期均线约1个月，长期均线约5个月 |
| **回看期 LB3** | 3周 | 动量计算窗口 |
| **偏离度 D15** | ≤15% | 价格偏离MA21上限 |
| **持仓数 H3** | 3只 | 最多同时持有3只ETF |
| **G3过滤** | 三周≥0% + 本周≥-1% | 避免弱势标的 |

## 三、选股规则

### 入场四条件（同时满足）

| # | 条件 | 计算方式 | 直觉理解 |
|:--:|------|---------|----------|
| 1 | **3周动量 > 0** | 当前价 / 3周前价 - 1 > 0 | 价格在涨 |
| 2 | **价格 > MA5 > MA21** | 5周均线 > 21周均线 | 均线多头排列 |
| 3 | **偏离度 ≤ 15%** | 当前价 / MA21 - 1 ≤ 0.15 | 没追高太远 |
| 4 | **G3过滤** | 三周涨幅≥0% 且 本周涨幅≥-1% | 避免弱势标的 |

### 排序与去重

1. 全部189只ETF计算动量，筛选满足三条件的标的
2. 按动量从高到低排序
3. **同板块去重**：同一指数类别只保留动量最高的1只
4. 取前3只

### 信号时序（无前视偏差）

```
周五收盘(T周)：计算信号 → 排名去重 → 生成候选名单
下周一开盘(T+1周)：按候选名单执行买卖
```

## 四、持仓管理

### 资金配置

- **总资金**：5万元
- **持仓上限**：3只
- **配置方式**：等权配置（每只占总额1/3）
- **换仓触发**：持仓不再满足入场三条件，或被更高动量标的替换

### 退出机制

#### ① 条件退出（每周检查）
持仓ETF**不再满足入场三条件中任一条**时，下周一卖出：
- 动量转负（近3周涨幅为负）
- 均线空头（价格 ≤ MA5，或 MA5 ≤ MA21）
- 偏离超限（偏离MA21 > 15%）

#### ② 止损退出（每周检查）
| 类型 | 条件 | 说明 |
|------|------|------|
| **成本止损** | 亏损 ≥ 8% | 相对买入价 |
| **高点回撤止损** | 从持仓最高点回撤 ≥ 10% | 锁利 |

> 回测中止损触发频率较低，主要退出方式是条件退出（动量转负/均线破坏）。

### 换仓判断流程

```
每周五收盘：
1. 计算189只ETF的动量/MA5/MA21/偏离度
2. 筛选满足三条件的标的
3. 同板块去重，按动量排序取Top3
4. 对比当前持仓：
   - 持仓不在Top3中 → 下周一卖出
   - 止损触发 → 下周一卖出
   - 空仓位有更优标的 → 下周一买入补仓
5. 无合格标的 → 空仓持现金
```

## 六、回测方法论

### 回测框架

- **脚本**：`D:\QClaw_Trading\backtest_v4.py`
- **方法**：净值法（无100股整数限制），等权再平衡
- **执行时点**：T周信号 → T+1周close执行（模拟周一开盘）
- **标的池**：189/194只ETF有本地历史数据（5只缺失）
- **数据范围**：日K线聚合为周K线，历史数据最早可追溯至2010年

### 回测局限

1. **周频止损**：实际应日频检查止损，周频会低估真实回撤
2. **无滑点/手续费**：736次买卖假设零成本，实盘需要扣除
3. **T+1 close ≠ T+1 open**：用close执行可能引入微小偏差
4. **数据不完整**：5只ETF缺失，所有数据截至2026-04-24

## 八、风险提示

1. **集中风险**：3只持仓意味着单只ETF暴雷影响占总仓位33%，远高于5只的20%
2. **2025年回撤-48%**：年内最大回撤接近策略设定上限，需要心理承受力
3. **周频检查的局限性**：实盘中如果某周内价格暴跌8%以上，等到周五才卖可能亏损更大
4. **回测vs实盘**：净值法忽略100股限制、滑点、停牌无法卖出等因素，实盘收益会打折
5. **2019-2025的大Beta**：过去6年A股整体beta偏正，动量策略天然受益，未来不一定

## 九、操作流程

### 每周五收盘后
1. 运行 `backtest_v4.py` 的筛选逻辑（或 `weekly_scan_v4.py` 实时扫描）
2. 获取Top3候选名单
3. 对比当前持仓，生成操作指令

### 下周一开盘
1. 执行卖出（条件不满足的持仓）
2. 执行买入（空仓位补入新标的）
3. 记录交易

### 每周
1. 检查止损条件（理想是每日检查，最低每周检查）
